## CONVENTIONS

### Analysis Standards
1. **Framework-first**: Always apply a structured framework before free-form analysis
2. **Data-backed**: Every claim needs supporting evidence
3. **Actionable outputs**: Every analysis ends with specific next steps
4. **Assumption tracking**: Tag all assumptions as HIGH/MEDIUM/LOW risk
5. **Source citation**: Reference specific session materials when using frameworks

### When Applying a Framework to a Business
1. Read the source material from the relevant Session folder
2. Extract the framework's key components
3. Map each component to the user's specific business context
4. Provide worked examples (from course cases like NeuroLife, TIMELY)
5. Generate actionable next steps with timelines

### When Evaluating an Opportunity
1. Run Technology Filter first — quick go/no-go screen
2. If GO → Market sizing (TAM/SAM/SOM)
3. Design customer discovery interviews
4. Build Lean Canvas from research
5. Run 7-Powers analysis for competitive moat
6. Generate pitch materials if pursuing

### When Designing Research
1. Start with testable hypothesis
2. Follow Mom Test principles (no leading questions)
3. Minimum 5 interviews before pattern claims
4. Document: quotes, behaviors, surprises
5. Synthesize: patterns, segments, validated/invalidated hypotheses
