## CONVENTIONS

### Analysis Standards
1. **Data-backed**: Every claim needs supporting evidence (market data, interview quotes, metrics)
2. **Framework-first**: Always apply a structured framework before free-form analysis
3. **Actionable outputs**: Every analysis ends with specific next steps
4. **Assumption tracking**: Explicitly state and tag all assumptions for validation
5. **Confidence levels**: Rate conclusions as HIGH / MEDIUM / LOW confidence with reasoning

### Research Standards
1. **Primary > Secondary**: Prioritize customer interviews and direct observation
2. **Minimum viable research**: At least 5 interviews before pattern claims
3. **Hypothesis-driven**: Every research effort starts with a testable hypothesis
4. **Anti-bias protocols**: Use open-ended questions, avoid leading, record contradictions

### Deliverable Standards
1. **Executive summary first**: Every deliverable starts with a 3-sentence summary
2. **Visual over verbal**: Use tables, canvases, and diagrams where possible
3. **So-what test**: Every data point needs an implication stated
4. **Decision-ready**: Deliverables should enable a specific decision

## WORKFLOW RULES

### When Evaluating a Technology/Opportunity
1. Run **Technology Filter** — is this worth pursuing at all?
2. Map the **market landscape** — who else is here?
3. Conduct **Quicklook Assessment** — what's the commercialization path?
4. Design **customer interviews** — validate with real users
5. Build **Lean Canvas** — model the business
6. Identify **7-Powers** — what's the defensible advantage?

### When Entering a New Market
1. **Jobs-to-be-Done analysis** — what are customers actually hiring for?
2. **Customer Discovery Interviews** (minimum 15) — validate jobs
3. **Competitive analysis** using 7-Powers + Porter's
4. **Value proposition design** — how do we win?
5. **Lean Canvas** — business model design
6. **Go-to-market plan** — channels, pricing, launch sequence

### When Pivoting or Repositioning
1. **Post-mortem** current approach — what's not working and why?
2. **Assumption audit** — which assumptions were wrong?
3. **Solutionstorming** — generate 10+ alternative approaches
4. **Technology Filter** on top 3 alternatives
5. **Lean Canvas** for chosen direction
6. **90-day experiment plan**

### When Preparing for Investors
1. **Market sizing** — TAM/SAM/SOM with methodology
2. **Competitive moat analysis** using 7-Powers
3. **Unit economics** — CAC, LTV, payback period
4. **Traction metrics** — growth rate, retention, NPS
5. **Financial projections** — 3-year model with assumptions
6. **Pitch deck structure** — problem, solution, market, traction, team, ask
